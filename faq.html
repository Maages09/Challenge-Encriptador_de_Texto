FAQs

1. ¿Qué función cumple esta página?
Esta página permite encriptar y desencriptar texto de manera sencilla utilizando un conjunto de reglas específicas. Es una herramienta diseñada para proteger la información y hacerla legible solo para quienes conocen el método de desencriptación.

2. ¿Qué tecnologías se utilizaron para desarrollar esta página?
La página fue desarrollada utilizando únicamente HTML y CSS. El objetivo fue demostrar cómo se puede crear una herramienta funcional sin necesidad de utilizar lenguajes de programación más avanzados.

3. ¿Por qué se eligió desarrollar este proyecto?
Este proyecto forma parte del curso Oracle Next Education, desarrollado en asociación con Alura Latam, como una manera de aplicar los conocimientos adquiridos en un desafío real. Es un ejemplo práctico de cómo se pueden resolver problemas con creatividad y conocimientos básicos en desarrollo web.

4. ¿Cómo se puede utilizar esta página para encriptar un mensaje?
Para encriptar un mensaje, simplemente ingresa el texto en el campo correspondiente y haz clic en el botón de "Encriptar". El texto será transformado siguiendo las reglas de encriptación predefinidas, y el resultado aparecerá en pantalla.
